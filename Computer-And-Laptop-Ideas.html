As an IT student, here are the features I prefer.

CPU | Intel Core i5-1235Uk

The Intel Core i5-1235U is a mid-range mobile processor for thin and light laptops. It has 10 cores (2 performance cores and 8 efficient cores) and 12 threads. It can run at up to 4.4 GHz on the performance cores and up to 3.3 GHz on the efficient cores. It has 12 MB of L3 cache and a base power consumption of 15 W. It also has an integrated graphics card called Intel Iris Xe Graphics with 80 execution units and a max frequency of 1.2 GHz. It supports various memory types, PCIe 4.0, and Intel vPro Essentials. It is based on the Alder Lake architecture and manufactured on Intel’s 10 nm process.

Video Graphics | Intel® Iris® Xe Graphics

Intel Iris Xe graphics are integrated graphics in some 11th generation Intel Core processors. They have better performance and features than previous Intel graphics, such as AV1 video decoding, HDR video, deep learning inference, and gaming capabilities. They are suitable for mainstream desktop and business PCsIntel Iris Xe graphics are part of Intel’s strategy to compete with other graphics card makers like Nvidia and AMD.

Storage | 512 GB PCIe® NVMe? M.2 SSD

A 512 GB PCIe® NVMe M.2 SSD is a type of solid state drive (SSD) that has a capacity of 512 gigabytes and uses the PCI Express (PCIe) interface with the Non-Volatile Memory Express (NVMe) protocol to communicate with the computer. It has a form factor of M.2, which is a small and thin card that can be inserted into a slot on the motherboard or a laptop

System| 64-bit operating system, x64-based processor

A 64-bit operating system, x64-based processor means that the computer has a CPU that can handle 64-bit instructions and an OS that can run 64-bit programs. This allows the computer to use more memory, perform faster calculations, and run more complex applications than a 32-bit system. A 64-bit system is also compatible with 32-bit programs, but not with 16-bit ones.
